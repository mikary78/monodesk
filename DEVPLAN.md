# MonoDesk 개발 계획서 (Development Plan)

문서 버전: v1.0
작성일: 2025년
작성자: MonoBound 대표 (Claude Code 협업)

---

## 1. 개발 원칙

```
① 모듈 하나씩 완성 후 다음 모듈로 진행
② 기본 기능 먼저 → AI 기능 나중에 추가
③ 매 모듈 완성 시 QA 검증 필수
④ 동작하는 결과물을 자주 확인
⑤ 문제 발생 시 즉시 멈추고 해결 후 진행
```

---

## 2. 전체 개발 단계

```
Phase 0  →  프로젝트 초기 설정 (완료 ✅)
Phase 1  →  핵심 모듈 개발 (현재)
Phase 2  →  AI 기능 추가
Phase 3  →  고도화 및 통합
Phase 4  →  마케팅 자동화 (별도 프로젝트)
```

---

## 3. Phase 0 — 프로젝트 초기 설정 ✅ 완료

```

```

## 4. Phase 1 — 핵심 모듈 개발

### Step 1. 프로젝트 기본 구조 생성
```
목표  : 개발 환경 세팅 및 기본 레이아웃 완성
산출물:
  - 프론트엔드 React 프로젝트 생성
  - 백엔드 FastAPI 프로젝트 생성
  - SQLite DB 연결
  - 사이드바 네비게이션 레이아웃
  - 라우팅 구조
  - 다크/라이트 모드

완료 기준:
  ✅ 브라우저에서 앱이 실행됨
  ✅ 사이드바 메뉴 클릭 시 화면 전환
  ✅ 다크/라이트 모드 전환 동작
```

### Step 2. 세무/회계 모듈 ← 현재 진행 중
```
목표  : 매출/지출 관리 및 손익 확인

2-1. 매출 관리
  - 일별 매출 입력 폼 (카드/현금/기타)
  - 매출 목록 (날짜별 정렬)
  - 월별 매출 합계

2-2. 지출 관리
  - 지출 입력 폼 (항목/금액/분류/날짜)
  - 영수증 이미지 업로드 (OCR은 Phase 2)
  - 지출 분류별 목록

2-3. 손익 현황
  - 월별 손익 계산 (매출 - 지출)
  - 손익 차트 (막대 + 꺾은선)
  - 원가율 계산

2-4. 지분 정산 시뮬레이션
  - 4명 지분 비율 입력
  - 순이익 기반 배분 금액 자동 계산
  - 급여 차감 후 배당 가능 금액

2-5. 세무사 리포트 Excel 출력
  - 월간 매출/지출 내역 Excel 생성

완료 기준:
  ✅ 매출/지출 입력 → 저장 → 목록 조회
  ✅ 손익 차트 정상 표시
  ✅ 지분 계산 정확성 검증
  ✅ Excel 파일 정상 생성
  ✅ QA 에이전트 검증 통과
```

### Step 3. 매출 분석 모듈
```
목표  : POS 데이터 기반 매출 분석

3-1. POS CSV 가져오기
3-2. 매출 트렌드 차트 (일/주/월)
3-3. 메뉴별 판매 분석
3-4. 요일/시간대 분석
3-5. 결제 수단 분석

완료 기준:
  ✅ CSV 파일 업로드 → 데이터 정상 파싱
  ✅ 모든 차트 정상 표시
  ✅ QA 에이전트 검증 통과
```

### Step 4. 재고/발주 관리 모듈
```
목표  : 식재료 재고 및 발주 관리

4-1. 재고 항목 등록/관리
4-2. 거래처 정보 관리
4-3. 발주 내역 기록
4-4. 재고 부족 알림

완료 기준:
  ✅ 재고 입출고 정상 반영
  ✅ 거래처별 발주 이력 조회
  ✅ QA 에이전트 검증 통과
```

### Step 5. 메뉴 관리 모듈
```
목표  : 메뉴 레시피 및 원가 관리

5-1. 메뉴 등록/수정/삭제
5-2. 레시피 등록 (재료 + 수량)
5-3. 원가/마진율 자동 계산
5-4. 계절 메뉴 관리

완료 기준:
  ✅ 메뉴 원가 계산 정확성 검증
  ✅ 계절 필터링 정상 동작
  ✅ QA 에이전트 검증 통과
```

### Step 6. 직원 관리 모듈

목표  : 직원 근태 및 급여 관리

6-2. 근태 기록

## 5. Phase 2 — AI 기능 추가

### AI Step 1. 영수증 OCR
```
- Tesseract 인쇄체 영수증 인식
- Ollama llava 수기 영수증 인식
- 지출 자동 입력 연동
```

### AI Step 2. 회의록 자동 정리
```
- faster-whisper 음성 → 텍스트
- Ollama 회의록 양식 자동 생성
- 법인 관리 모듈 연동
```

### AI Step 3. 매출 분석 AI 리포트
```
- Ollama 기반 자연어 경영 인사이트
- 대시보드 연동
```

### AI Step 4. 문서 자연어 검색
```
- ChromaDB 벡터 DB 구축
- 법인 문서 검색 기능
```
