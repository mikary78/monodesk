# 데일리 단가 추적 모듈 추가 — 엑셀 4.데일리단가원재료 시트 구현

## 변경 요약
- `daily_price_records` 테이블 신규 생성
- `inventory_items` 테이블에 `is_daily_price_tracked` 컬럼 추가
- 월별 그리드 조회, 단가 기록 UPSERT, 품목 추적 토글 API 추가
- `DailyPriceGrid.jsx` 엑셀형 그리드 UI 구현
- `InventoryPage`에 "데일리 단가" 탭 추가

## 배경
수산물 등 시가 품목은 매일 단가가 변동되므로
일별 단가를 추적하여 원가율 변동을 모니터링하기 위해 구현.
엑셀 4.데일리단가원재료 시트를 시스템화.

## 주요 설계 결정
- **UPSERT 방식**: `item_id + record_date` UNIQUE 제약으로 중복 방지, 같은 날 재입력 시 업데이트
- **amount 자동계산**: 서비스 레이어에서 `quantity × unit_price` 계산 후 저장
- **단가 등락 표시**: 직전 기록 역탐색 방식 — 당일 이전 가장 최근 기록과 비교 (상승=빨강, 하락=파랑)
- **라우터 순서**: `/daily-price/summary/{year}/{month}` 고정 경로를 `/daily-price/{year}/{month}` 동적 경로보다 먼저 등록

## 기능 설명
- 좌측 품목명 + 날짜 헤더 교차 그리드 (가로 스크롤)
- 셀 클릭 → 팝오버에서 수량/단가/구매처 입력
- 우측 품목별 월합계, 하단 날짜별 일합계
- "추적 품목 설정" 모달로 추적 대상 ON/OFF 토글
