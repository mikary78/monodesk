# 월초/월말 재고 스냅샷 모듈 추가 — 엑셀 8-1/8-2 시트 구현

## 변경 요약
- `inventory_snapshots` 테이블 신규 생성 (UNIQUE 제약: snapshot_type+year+month+item_id)
- 월말 확정 시 다음달 월초로 자동 이월 로직 구현 (`_copy_month_end_to_next_start`)
- `InventorySnapshot.jsx` 컴포넌트: 카테고리별 테이블, 인라인 편집, 월 네비게이션, 확정 처리
- `InventoryPage.jsx`에 "월초재고" / "월말재고" 탭 2개 추가

## 배경
엑셀 8-1.월초재고 / 8-2.월말재고 시트의 업무 프로세스를 웹앱으로 전환:
- 월말에 실사 후 월말재고 입력 → 확정
- 확정된 월말재고 → 다음달 월초재고로 자동 이월
- 사용량 추정 공식: 월초재고 - 월말재고 + 매입 = 당월 사용량

## 주요 설계 결정
- **UNIQUE(snapshot_type, year, month, item_id)**: 동일 월 동일 품목 중복 방지
- **확정(is_confirmed=1) 후 수정 불가**: 서비스 레이어에서 강제 검증
- **month_end 확정 시 자동 이월**: `_copy_month_end_to_next_start()` 내부 함수로 분리
- **초안 생성**: `inventory_items.current_quantity` 현재 재고 기준, 미확정 상태 덮어쓰기 가능
- **라우터 경로 순서**: `/snapshot/confirm` → `/snapshot/generate/...` → `/snapshot/{type}/{year}/{month}` 순으로 등록 (고정 경로 우선)
- **year/month 자체 관리**: `InventorySnapshot` 컴포넌트가 `initialYear/initialMonth` props를 받되 내부 state로 관리 (월 네비게이션 지원)

## API 엔드포인트 요약
| 메소드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/inventory/snapshot/{type}/{year}/{month}` | 스냅샷 조회 (자동 이월 포함) |
| POST | `/api/inventory/snapshot/confirm` | 스냅샷 확정 처리 |

| PUT | `/api/inventory/snapshot/{id}` | 항목 수량/단가 수정 |
