## 구현 기능

### 1. 동업자 현황 탭
- 동업자 CRUD (추가/수정/삭제)
- 지분율 합계 100% 초과 방지 검증
- 기본 동업자 4명 초기 등록 seed 기능
- 지분율 시각화 바, 역할/연락처/계좌 정보 관리

### 2. 배당 정산 탭
- 연간 순이익 + 배당 비율 입력 → 동업자별 배당금 즉시 계산 (시뮬레이션, DB 저장 없음)
- 정산 확정 시 DB 저장 (동일 연도 기존 기록 upsert 처리)
- 지급 상태 토글 (미지급 ↔ 지급 완료), 지급일 자동 기록
- 연도별 배당 기록 전체 삭제

### 3. 법인 비용 탭
- 법인 비용 CRUD (세무사비, 법인보험료, 등기비 등)
- 월/분류별 필터링, 반복 비용 플래그, 금액 합계 표시

### 4. 재무 개요 탭
- 연간 KPI 4종: 매출/총지출/순이익/순이익률
- 전년 대비 순이익 증감률 계산
- 동업자별 예상 배당금 (100% 배당 기준, 지분율 시각화 바)
- 법인 비용 분류별 Recharts 파이 도넛 차트
- 연간 손익 요약 테이블

## DB 구조 핵심 설계 결정
- `dividend_records`: UNIQUE(year, partner_id) → 연도별 동업자 배당 1건만 허용, upsert 처리
- `partner_name` 스냅샷 컬럼: 동업자 이름 변경/삭제 후에도 배당 이력 정확히 보존
- `equity_ratio_snapshot`: 배당 정산 시점 지분율 이력 보존

## 테스트 및 검증 방법
1. 백엔드 서버 재시작: `uvicorn main:app --reload --port 8000`
2. Swagger UI 확인: http://localhost:8000/docs#/법인 관리
3. 기본 동업자 초기화: POST /api/corporate/partners/seed
4. 배당 시뮬레이션: POST /api/corporate/dividend/simulate
5. 재무 개요 확인: GET /api/corporate/overview?year=2026
6. 프론트엔드 사이드바 "법인 관리" 메뉴 클릭 → 4개 탭 동작 확인
