## 주요 기능
1. **메뉴 카테고리** — CRUD, 색상/정렬 순서 설정, 카테고리에 메뉴 있을 시 삭제 방지
2. **메뉴 아이템** — CRUD, 판매/중지 토글, 대표 메뉴 설정, 카테고리/상태 필터
3. **원가 자동 계산** — 구성 재료 추가/수정/삭제 시 메뉴 원가 실시간 재계산
4. **원가율 분석** — 구간별 분포 파이 차트, 고마진/저마진 순위, 평균 원가율
5. **KPI 카드** — 전체 메뉴 수, 평균 원가율, 원가율 경고 메뉴, 대표 메뉴 수

## 비즈니스 로직
- 원가율 = 원가 ÷ 판매가 × 100
- 마진 = 판매가 - 원가
- 원가율 구간: 0~30%(고마진/초록), 30~50%(양호/파랑), 50~70%(주의/노랑), 70%+(경고/빨강)
- 구성 재료 소계 = 수량 × 단가
- 구성 재료 합산 = 메뉴 원가로 자동 반영

## 테스트 방법
1. 백엔드 서버 실행: `cd backend && uvicorn main:app --reload`
2. `/api/menu/seed-categories` POST 호출로 기본 카테고리 생성
3. Swagger UI(`/docs`)에서 API 동작 확인
4. 프론트엔드: `cd frontend && npm run dev`로 실행 후 사이드바 "메뉴 관리" 클릭
