### 수정된 파일
- `frontend/src/pages/OperationsPage.jsx` : "일일마감", "이슈관리" 탭 추가 (7개 탭)

### 데이터베이스 변경
- `daily_closings` 테이블 신규 생성 (17개 컬럼)
  - 권종별: bill_100000 ~ coin_100 (7개)
  - 자동계산: total_cash, prev_day_cash, balance
- `daily_issues` 테이블 신규 생성 (8개 컬럼)
  - issue_type: customer / ingredient / employee

## 핵심 비즈니스 로직
- `prev_day_cash` = 전날 closing 레코드의 balance 자동 참조
- `total_cash` = 권종 수량 × 단위금액 합계
- `balance` = prev_day_cash + daily_deposit - daily_expense

## 테스트 및 검증 방법
- [ ] 운영관리 → 일일마감 탭 진입
- [ ] 권종 수량 입력 시 우측 금액 자동계산
- [ ] "현금 합계" 실시간 갱신
- [ ] 전일 이월 값이 전날 저장된 잔액과 일치
- [ ] 저장 후 "시재 수정" 버튼으로 전환
- [ ] 이슈관리 탭: 등록/완료토글/처리내역 인라인 편집/삭제

## 주의사항
- 전일 데이터가 없으면 이월 = 0 (정상 동작)
- 이슈 삭제는 하드 삭제 (복구 불가)

## 향후 작업 제안
- 월별 시재 현황 리포트 화면 추가 가능
- 이슈 통계 (유형별 건수, 미완료 현황) 대시보드 연계 가능
