# 변경 이력: 로그인 및 역할 기반 권한 관리 시스템

## 기본 정보
- 날짜: 2026년 04월 13일
- 브랜치: feature/auth-system
- 작업자: Claude Code

## 요청 내용
MonoDesk 프로젝트에 3가지 역할(admin/manager/staff) 기반의 로그인 및 권한 관리 시스템 구현.
기존 모든 API 엔드포인트에 역할 기반 접근 제어를 적용하고,
프론트엔드에 로그인 페이지와 JWT 인증 상태 관리를 추가.

## 변경한 내용

### 추가된 파일
**백엔드**

## 기술 결정 사항
- JWT 토큰 만료: 8시간 (ACCESS_TOKEN_EXPIRE_HOURS)
- bcrypt 버전: 4.0.1 고정 (passlib 1.7.4와 호환성 문제로 5.x 제외)
- 토큰 저장: localStorage (access_token, auth_user)
- 로그아웃: 클라이언트 토큰 삭제 + 서버 알림 (서버는 Stateless)
- 401 응답: apiClient.js에서 자동 감지 → 토큰 삭제 → /login 리다이렉트

## 주의사항
- bcrypt==4.0.1 버전 고정 필수 (requirements.txt에 명시됨)
- SECRET_KEY는 로컬 전용이므로 하드코딩. 외부 서비스 연동 시 환경변수로 교체 필요
- admin 기본 비밀번호(admin1234)는 최초 로그인 후 변경 권장
- documentApi.js의 BASE 경로를 상대경로(/api/documents)에서 절대경로(http://localhost:8000/api/documents)로 변경함
