# 근무표 달력 모듈 추가

## 변경 요약
- attendance_records 테이블에 daily_status 컬럼 추가 (마이그레이션 012)
- AttendanceStatus Enum 정의 — 9가지 근태 상태
- GET /api/employee/attendance/monthly-calendar 엔드포인트 추가
- PATCH /api/employee/attendance/{id}/status 엔드포인트 추가
- AttendanceCalendar.jsx 신규 구현 (달력형 근무표 UI)
- EmployeePage.jsx에 "근무표 달력" 탭 추가 (4번째 탭)

## 배경
엑셀 2-2.근무표 시트 기준으로 전 직원의 월별 근태를 한눈에 확인하고 셀 클릭으로 빠르게 수정할 수 있는 달력형 UI 구현

## 변경 파일
- `backend/migrations/012_attendance_status.sql` (신규)
- `backend/models/employee.py` (AttendanceStatus Enum + daily_status 컬럼)
- `backend/schemas/employee.py` (AttendanceCalendarResponse 스키마)
- `backend/routers/employee.py` (monthly-calendar, PATCH status 엔드포인트)
- `backend/services/employee_service.py` (get_monthly_calendar, update_attendance_status)
- `frontend/src/api/employeeApi.js` (getMonthlyCalendar, updateAttendanceStatus)
- `frontend/src/components/modules/employee/AttendanceCalendar.jsx` (신규)
- `frontend/src/pages/EmployeePage.jsx` (근무표 달력 탭 연결)

## AttendanceStatus Enum (9가지)
| 코드 | 표시 | 색상 |
|------|------|------|
| 근무 | 근 | 초록 |
| 휴무 | 휴 | 회색 |
| 월차 | 월 | 파랑 |
| 반차 | 반 | 하늘 |
| 결근 | 결 | 빨강 |
| 조퇴 | 조 | 주황 |
| 권장휴무 | 권 | 보라 |
| 지원 | 지 | 노랑 |
| (미입력) | — | 없음 |

## UI 동작
- 가로축: 날짜(1~말일), 세로축: 직원
- 셀 클릭 → 드롭다운으로 상태 선택 → 즉시 PATCH 요청
- 하단 집계 행: 직원별 근무일수 / 휴무일수 표시
- 월 네비게이션(이전/다음) 지원
