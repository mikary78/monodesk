# 직원 상세정보 확장 — 근무파트/식대/차량유지비/계약형태

- employees 테이블에 work_part, meal_allowance, car_allowance, work_condition, contract_type 컬럼 추가
- 계약형태(4대보험/3.3%/시급알바)에 따른 급여 계산 분기 로직 추가
- EmployeeFormModal에 신규 필드 입력 UI 추가

## 급여 계산 변경
| 계약형태 | 공제 방식 |
|---------|---------|
| 4대보험 | 과세급여(총급여-식대-차량유지비) 기준 4대보험 |
| 3.3% | 총급여 × 3.3% 원천징수 |
| 시급알바 | 공제 없음 |

### 하위 호환 처리
- `contract_type`이 없는 기존 데이터: `has_insurance` 값으로 fallback
  - has_insurance=True → "4대보험"으로 처리
  - has_insurance=False → "시급알바"로 처리

## 법적 근거
- 식대/차량유지비 비과세: 소득세법 제12조
- 3.3% 원천징수: 소득세법 제129조 (소득세 3% + 지방소득세 0.3%)
- 4대보험 요율: 국민연금 4.5%, 건강보험 3.545%, 장기요양 12.95%(건강보험 기준), 고용보험 0.9%
